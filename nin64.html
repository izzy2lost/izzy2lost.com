Nin64

Nintendo 64 emulator for Android

Coming soon

Nin64 is a Nintendo 64 emulator for Android powered by the Mupen64Plus-Next libretro core with GLideN64 hardware rendering.

Key Features

  • Mupen64Plus-Next libretro core with GLideN64 renderer
  • Hardware-accelerated graphics via OpenGL ES 3.0
  • Resolution scaling up to 8x native for a sharper image
  • Texture pack support — load custom high-resolution texture packs
  • Cheat manager — browse and toggle per-game cheats (GameShark / Action Replay)
  • Configurable touch controls — reposition, resize, and remap the on-screen layout
  • Physical controller remapping — fully remap USB and Bluetooth controller buttons and sticks
  • Game library browser — scan and launch ROMs from local storage
  • Save states — save and restore progress at any point
  • In-game save support (SRAM, FlashRAM, EEPROM, MemPak)
  • Low-latency audio via Google Oboe
  • ARM64 and x86_64 ABI support
  • Free to play with ads — remove ads with a one-time in-app purchase

Coming Soon

  • Cloud save sync
  • Online multiplayer

Device Requirements

  • Android 8.0+ (API 26 or newer)
  • ARM64 (arm64-v8a) or x86_64 device
  • OpenGL ES 3.0 capable GPU
